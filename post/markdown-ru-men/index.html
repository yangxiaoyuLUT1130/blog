Markdown入门

2019-09-17 / 2 min read

在平时的文字记录时，office等一些“富文本”软件，用于记录自己的学习笔记时，会使得自己很大一部分时间用于版本，并且对于数学公式的支持也不够友好。Mrkdon是一种“标记语言”，语法相比于html而言，十分简洁。易于学习，记录效率高，能够让你专注于写作本身。

Markdown

We believe that writing is about content, about what you want to say - not about fancy formatting.
我们坚信写作写的时内容，是所思所想，而不是花样的排版格式。
—— Ulysses for Mac

Markdown产生的目的在于让记录写作者能偶专注于写作本身，而非将精力注重于样式排版；其语法简洁、高效，对于代码、数学公式的支持友好。Markdown 从写作到完成，导出格式随心所欲，你可以导出HTML格式的文件用来网站发布，也可以十分方便的导出PDF格式。
这里你可以看到官方的Markdown语法规则。

  • 创始人John Gruber的Markdown的语法说明

这里除了介绍基本的语法文字的语法外，重点将介绍如何使用Markdown输入数学公式。

文字

需要提前说明的是，在Markdown中，文字之间的空格在输出显示时时将被忽略的。如果是要进行分段，则需要“空一行”来完成。

列表

列表有有序列表和无序列表两种。

无序列表

* 好好学习
* 天天向上

*后要留有空格，在接上文字。

输出显示效果如下：

  • 好好学习
  • 天天向上

有序列表

\1. 第一
\1. 第二
\1. 第三

效果如下：

  1. 第一
  2. 第二
  3. 第三

注意，有序列表时，与你输入的数字大小是无关的，只要是“
数字 + 点”就行。

引用

当你引用一小段别处的句子，则需要使用引用格式。使用>符号，加一个空格的形式。

表格

表格的输出采用|来完成，注意第二行的----以及:可以用于表格的对齐方式控制。

| Tables | Are |Cool |
|---------------|:---------------|--------:|
| col 3 is | right-aligned | $600 |
| col 2 is | centered | $12 |
| zebra stripes | are neat | $1 |

输出效果如下：

Tables Are Cool
col 3 is right-aligned $600
col 2 is centered $12
zebra stripes are neat $1

代码框的使用

在word中，代码的输入是让人十分头疼的，主要是因为字体的原因，使得代码在word中的可读性十分差；而Markdown中，对于代码的支持十分友好，易于阅读。当需要记录代码时，使用`符号进行。

行内代码采用代码前后各一个`。

例如：

采用C输入Hello, World!。printf("Hello, World!\n");。这printf函数时打印函数。

当需要进行整段的代码书写时，采用三个`使用代码框。

```具体的语言（如C就写C，python就写python），这样能够开启代码高亮
代码................................
................................
```
例如：
这时C语言代码：

int main(void)
{
    printf("Hell, World!\n");
	return 0;
}

这是Python的代码段：

def test():
    print('Hello, World!')
