Markdown入门

2019-09-17 / 12 min read

在平时的文字记录时，office等一些“富文本”软件，用于记录自己的学习笔记时，会使得自己很大一部分时间用于排本，并且对于数学公式的支持也不够友好。Mrkdon是一种“标记语言”，语法相比于html而言，十分简洁。易于学习，记录效率高，能够让你专注于写作本身。

Markdown

We believe that writing is about content, about what you want to say - not about fancy formatting.
我们坚信写作写的是内容，是所思所想，而不是花样的排版格式。
—— Ulysses for Mac

Markdown产生的目的在于让写作者能够专注于写作本身，而非将精力注重于样式排版；其语法简洁、高效，对于代码、数学公式的支持友好。Markdown 从写作到完成，导出格式随心所欲，你可以导出HTML格式的文件用来网站发布，也可以十分方便的导出PDF格式。
这里你可以看到官方的Markdown语法规则。

  • 创始人John Gruber的Markdown的语法说明

这里除了介绍基本的语法文字的语法外，重点将介绍如何使用Markdown输入数学公式。

需要提前说明的是，在Markdown中，文字之间的空格在输出显示时时将被忽略的。如果是要进行分段，则需要“空一行”来完成。

代码框的使用

在word中，代码的输入是让人十分头疼的，主要是因为字体编码的原因，使得代码在word中的可读性十分差；而Markdown中，对于代码的支持十分友好，易于阅读。当需要记录代码时，使用`符号进行。

行内代码采用代码前后各一个`。

例如：

采用C输出Hello, World!。printf("Hello, World!\n");。printf函数是打印函数。

当需要进行整段的代码书写时，采用三个`使用代码框，结束时也使用三个`。

```具体的语言（如C就写C，python就写python），这样能够开启代码高亮
代码................................
................................
```
例如：
这是C语言代码：

int main(void)
{
    printf("Hell, World!\n");
	return 0;
}

这是Python的代码段：

def test():
    print('Hello, World!')

latex公式

Markdown对于数学公式的支持是需要前端支持的，但是大多数都与latex语法相同。Markdown中数学公式的学习主要是各种数学符号的输入学习。

分数：

\frac{}{}，第一个花括号为分子，第二个花括号为分母
12\frac{1}{2}21​的输入为：\frac{1}{2}

上、下标

x_{a+b} 为xa+bx_{a + b}xa+b​。x^(a + b)为xa+bx^{a + b}xa+b。注意，上、下表中的元素需要用{}括住，当只有一个元素时，可以省略。这里a+b是3个元素，没有括号时，只有a会被识别。

运算符号
\cdot 为点乘符号，⋅\cdot⋅
乘号为\times ×\times×
\mathrm{}一般用于微分符号d，因为d不是一个变量，其次与变量d进行区别。
一重积分\int_{a} ^{b} x ^ {2} + 1 \mathrm{d} x
输出显示为∫ab(x2+1)dx\int_{a} ^ {b} (x ^ {2} + 1) \mathrm{d} x∫ab​(x2+1)dx

二重积分为\iint
求和符号\sum
偏导符号\partial
无穷大为\infty
小空格 \
4个空格\quad
